Security Policy

V1.4 AUDITED

Our Hardening Standards

  • Encryption: All S3 buckets and RDS instances are encrypted using AWS KMS (CMKs).
  • Network Isolation: Zero public access to database subnets. NAT Gateway-only outbound traffic for private subnets.
  • IAM Polices: We strictly follow the Principle of Least Privilege (PoLP) across all 42 Terraform resources.
