Deployment Manual

Step 1: Configure Backend

Edit the providers.tf file to point to your secure S3 backend for state locking.

terraform {
  backend "s3" {
    bucket = "ranasinghe-infra-state"
    key = "enterprise/terraform.tfstate"
  }
}
